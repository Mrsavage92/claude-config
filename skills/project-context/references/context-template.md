# CONTEXT.md Template

Use this template when writing `{project-root}/CONTEXT.md`. Fill every section. If a section has no source, write "MISSING — flag for human" and list it in section 10.

---

## Output Format

```markdown
# Project Context — {Product Name}

**Slug:** `{slug}`
**Generated:** {ISO date}
**Sources read:** {N}
**Validator verdict (if applicable):** {BUILD | VALIDATE-FIRST | KILL | N/A}
**Revenue status:** {LIVE-WITH-PAYING-CUSTOMERS | PRE-REVENUE-VALIDATED | PRE-REVENUE-UNVALIDATED | N/A}
**TTL:** 7 days

---

## 1. Product Goal
One sentence. Plain English. What this product IS — not "leverages AI to empower".

Source: `{file}` lines {x-y}

---

## 2. ICP (Ideal Customer Profile)

- **Industry:** {specific vertical}
- **Size:** {employee count, revenue band}
- **Geography:** {country/region}
- **Role:** {decision-maker title}
- **Pain trigger:** {what makes them search for this}

Source: `{file}`

If docs say "small businesses" or similar vague — write "TOO VAGUE — flag for human."

---

## 3. Value Proposition

- **Headline:** {one-liner positioning}
- **Sells:** {what they pay for, concretely}
- **Vs. alternatives:** {how it differs from competitors named in docs}
- **Proof structure:** {how trust is established — case studies, audits, demos, etc.}

Source: `{files}`

---

## 4. Locked Decisions

Items that downstream skills MUST NOT re-litigate. Pulled from:
- CLAUDE.md Section D (highest priority — running ledger)
- `*Business_Blueprint*`
- `*Pricing_and_Scope_Rules*`
- `*Decisions*` sections of any doc

Format: `- [date if available] {decision}. Source: {file}`

Example:
- 2026-05-13 ICP shift to SMB-primary, global, USD pricing. Source: CLAUDE.md Section D, decision 18
- Free L1 scan locked as funnel entry. Source: CLAUDE.md Section D, decision 20

---

Source: `{file}`

## 8. Recent Activity (last 30 commits)

```
{commit hash} {date} {message}
...
```

## 9. Anti-Goals

What this product is explicitly NOT. Critical for preventing template drift.

Examples:
- NOT a "general AI website audit" — anchored to 9 specific suites with deterministic engine
- NOT competing with $5/mo SaaS — premium positioning
- NOT for agencies (anymore, per decision 18)

Source: CLAUDE.md Section D, `*Blueprint*`

If docs don't state anti-goals → "MISSING — flag for human. Loop may drift toward generic SaaS without this."
```
