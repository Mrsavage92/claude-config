# Per-Platform Checks & Rubrics

Use this reference when scraping and scoring each platform in Phase 1.3 of the audit. Each platform has unique data points and benchmarks.

---

## LinkedIn (Company Page)

**Best for:** B2B, SaaS, agencies, professional services, recruiting, thought leadership

**Scrape these signals:**
- Follower count + employee count displayed
- About section length + keyword quality
- Tagline/headline
- Website link + working?
- Industry + specialties listed
- **Featured section** — posts/pages pinned to top
- **Page admins visible** (shows leadership investment)
- Last 20 posts: date, format (text / image / video / article / document / poll / event), likes, comments, reposts
- Employee advocacy signal: how many employees list the company + share/engage with posts
- Commentary posts (reshares with original commentary) vs pure reshares
- Hashtag use patterns
- Comment reply rate (brand replying to commenters)

**Platform-specific strengths to score:**
- Document posts (PDF carousels) — high-reach format currently
- Native video vs YouTube links (native gets more reach)
- Polls usage (engagement amplifier)
- Newsletter presence
- Event listings

**Benchmarks:**
- Cadence: 3-5 posts/week for active company page
- Engagement rate: 2-3% for B2B
- Video share: 20%+ of posts should be video or document
- Employee advocacy: 5%+ of employees should engage with company content

## Instagram (Business Profile)

**Best for:** B2C, DTC e-commerce, lifestyle, creators, visual brands, food, beauty, fashion, fitness

**Scrape these signals:**
- Follower count + following count + post count
- Bio: text, keywords, CTA, link stack (Linktree/Beacons/native)
- Profile category (Business/Creator)
- Verified blue tick status
- Story Highlights count + covers (branded or default?)
- Grid aesthetic: last 9 posts visual cohesion
- Pinned posts (top 3 slots)
- **Reels tab** — count of Reels vs feed posts
- **Guides tab** — present and populated?
- Shop tab — products tagged, Checkout-on-IG enabled
- Last 30 posts: format (static/carousel/Reel/Story-to-feed), caption, hashtag count, likes, comments
- Stories currently visible (24h window)
- **Story Highlights content** — organised by topic (product/team/testimonials/FAQ)?

**Platform-specific strengths to score:**
- Reels ratio (platform pushes Reels 10x more than static currently)
- Carousel ratio (save-friendly format, high reach)
- Story cadence (daily = engaged audience)
- Highlight organisation (functional FAQ replacement)
- Branded hashtag presence
- Collab posts (dual-author posts, 2x reach)

**Benchmarks:**
- Cadence: 4-7 feed posts/week + daily Stories
- Engagement rate: 1-3% for <100k followers, 0.5-1% for larger
- Reels share: 60%+ of posts should be Reels in 2025+
- Story highlights: 5+ organised highlights

## TikTok

**Best for:** Consumer brands, creators, education, entertainment, Gen Z/Millennial reach, trending-led marketing

**Scrape these signals:**
- Follower count + following count + likes count + video count
- Bio: text, keywords, link in bio
- Pinned videos (3 slots)
- Verified status
- TikTok Shop setup (for e-commerce) — badge visible
- Last 30 videos: date, views, likes, comments, shares, caption, hashtags
- **Audio strategy** — trending / original / brand
- Video duration distribution (short <15s / mid 15-60s / long >60s)
- Content themes
- Collab/duet/stitch participation
- **LIVE activity** — going live regularly?
- **TikTok Shop** — products linked in videos (for product brands)

**Platform-specific strengths to score:**
- Posting frequency (daily minimum for growth)
- Hook strength (first 3 seconds must retain)
- Trending audio usage (vs original — trend-riding amplifies reach)
- Caption hooks + CTAs
- Hashtag strategy (niche + broad mix, avoid spray)
- Duets/stitches (community engagement signal)
- Retention indicators (rewatch-friendly loops)

**Benchmarks:**
- Cadence: 1-3 videos/day for growth, min 3-5/week for presence
- Engagement rate: 5-9%
- View:follower ratio: 5-10x (TikTok's algo reaches non-followers)
- Hashtag count: 3-5 relevant, not 20+

## YouTube

**Best for:** Long-form content, SaaS demos, tutorials, product reviews, education, creators, searchable evergreen content

**Scrape these signals:**
- Subscriber count + total video count + total views
- Channel description + keywords
- Channel trailer (present for non-subs?)
- Featured video (present for subs?)
- Playlists: count, organised, named clearly
- Upload cadence (last 30 videos)
- **Shorts vs long-form split**
- Video titles: SEO-optimised or clickbait or descriptive?
- Thumbnails: custom + on-brand or auto-generated?
- Description quality: keywords, timestamps, links?
- Closed captions present?
- End screens + cards used?
- Community tab activity (polls, posts)
- Memberships enabled? (for creators)
- **Shopping shelf** populated? (for product channels)
- Super Thanks / Super Chat enabled?

**Platform-specific strengths to score:**
- Shorts integration (YT pushes Shorts algorithmically)
- Playlist architecture (SEO + binge-watch hack)
- Title SEO (front-loaded keywords)
- Custom thumbnails + consistency
- Description depth (timestamps = retention boost)
- End screen usage (watch-time stacking)
- Community tab (off-video engagement)

**Benchmarks:**
- Cadence: 1-2 long-form/week or 1 long-form + daily Shorts
- Engagement rate: 4-5% (likes+comments / views)
- Subscriber:view ratio: healthy channels get 5-10% of views from subs
- Shorts:long ratio: 5:1 if using Shorts as funnel

## Facebook (Page)

**Best for:** Local business, community groups, older demographic (35+), event-driven, B2C services, marketplaces

**Scrape these signals:**
- Page likes + followers (often diverge)
- Category + About section completeness
- Business info: address, hours, phone, email, website
- **Review rating** (if services category) — note this overlaps with reputation-audit, do not double-score
- Services tab populated?
- Shop tab (for products)
- Events tab — upcoming events listed?
- Last 20 posts: format, engagement, recency
- **Groups** linked or owned?
- Messenger auto-reply enabled?
- Community tab engagement

**Platform-specific strengths to score:**
- Complete business info (hours, address, phone)
- Events usage (Facebook events still drive local discovery)
- Groups (owned community = retention moat)
- Video content ratio (Meta prioritises video cross-platform)
- Shop setup (for e-comm)
- Messenger setup (direct sales channel)

**Benchmarks:**
- Cadence: 3-5 posts/week
- Engagement rate: 0.5-2% (Facebook organic reach is low)
- Video share: 40%+
- Reply to messages: <24h

## X / Twitter

**Best for:** Tech, SaaS, media, creators, real-time commentary, customer service, thought leadership

**Scrape these signals:**
- Follower count + following count + post count
- Bio text + location + website + pinned tweet
- Verified status (Blue / Gold / Grey)
- List memberships (signal of authority)
- Last 30 tweets: format (text/image/video/thread/quote), likes, retweets, replies, views (if visible)
- **Reply rate** — do they reply to other accounts or just broadcast?
- Thread usage
- Quote-tweet commentary
- Community notes attached to any posts?
- Space activity (audio)

**Platform-specific strengths to score:**
- Thread writing (high-reach format)
- Reply culture (1:1 engagement builds following)
- Pinned tweet strategy (top-converting offer / intro / best thread)
- Quote tweet commentary (vs pure retweets)
- Image attachment ratio (images 2x engagement)
- Bio link + CTA

**Benchmarks:**
- Cadence: 1-3 tweets/day for active presence
- Engagement rate: 0.5-1% (X has low organic reach)
- Reply:post ratio: 1:3 ideal (heavy engagement, light broadcast)

## Pinterest

**Best for:** E-commerce, home, beauty, food, wedding, DIY, fashion, visual discovery, traffic-to-site

**Scrape these signals:**
- Monthly views + followers
- Profile claimed + website verified?
- Bio + profile picture
- **Board architecture** — count, named clearly, on-brand covers?
- **Rich Pins enabled** (product / article / recipe pins pull metadata)
- Last 30 Pins: type (image/video/Idea Pin), saves, comments
- Idea Pins (Pinterest's Stories equivalent) used?
- Board covers custom + on-brand?
- Pin descriptions: SEO-optimised with keywords?
- Website traffic indicators (if business profile)

**Platform-specific strengths to score:**
- Board count + organisation (SEO-style categorisation)
- Rich Pins (required for e-commerce)
- Idea Pins (Pinterest pushes these)
- Keyword-stuffed descriptions (this is okay on Pinterest)
- Pin design consistency (branded overlays)
- Pin-to-product pipeline (for e-comm)

**Benchmarks:**
- Cadence: 5-15 Pins/week (Pinterest rewards consistency)
- Monthly views: should be 10-100x follower count for discovery platform
- Engagement via saves (not likes): >1% save rate

## Threads / Bluesky (Emerging)

**Best for:** Early adopters, text-first communities, Twitter alternatives, creators hedging platform risk

**Scrape these signals (Threads):**
- Follower count
- Bio + profile cross-link to IG
- Last 20 threads: engagement, format
- Cross-posting from other platforms detected?

**Scrape these signals (Bluesky):**
- Follower count
- Handle format (`.bsky.social` or custom domain)
- Starter packs membership
- Last 20 posts

**Score simply:**
- Present + posting regularly: +5 bonus to Presence Breadth
- Handle claimed but dormant: neutral (defensive claim is fine)
- Absent: no penalty unless brand is in tech/creator space where early-mover signals matter

## Handle Consistency Check

Test the same handle across all 8 platforms. Tools: direct URL check.

Example for `@glossbeauty`:
- `linkedin.com/company/glossbeauty` ✓ / ✗
- `instagram.com/glossbeauty` ✓ / ✗
- `tiktok.com/@glossbeauty` ✓ / ✗
- `youtube.com/@glossbeauty` ✓ / ✗
- `facebook.com/glossbeauty` ✓ / ✗
- `x.com/glossbeauty` ✓ / ✗
- `pinterest.com/glossbeauty` ✓ / ✗
- `threads.net/@glossbeauty` ✓ / ✗

Flag:
- Unclaimed + relevant to audience = risk of squatting (🔴 Fix Immediately — claim defensively, 10 min)
- Claimed by unrelated account = brand risk (🟠 Fix This Month — attempt reclaim process)
- Inconsistent handles across platforms = brand confusion (🟠 Fix This Month — document and unify where possible)
