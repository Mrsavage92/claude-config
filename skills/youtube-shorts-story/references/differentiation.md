# Differentiation: Own an Identity, Don't Wear a Clone's Skin

The measured corpus (reference-corpus.md) teaches the PHYSICS of retention. This file governs the line between adopting physics (safe, correct) and copying a creator's identity (clone-perception + originality-signal risk). Read this before ideation and before writing any generation prompt.

## The universal / signature split

| UNIVERSAL - adopt freely (retention physics) | SIGNATURE - never copy (a creator's identity) |
|---|---|
| Premise-drop opens | A specific creator's render look (e.g. soft-3D anatomy) |
| Chained but/therefore/so/until causality | A specific creator's vocal cadence/voice |
| Hot audio -11 LUFS, SFX-forward, ~2.9 w/s VO | A specific creator's recurring topic rotation |
| Continuous flow, 0-3 cuts, loop-back endings | A creator's channel name/branding/thumbnail style |
| Claim titles with a curiosity gap | Any creator's script wording |

If a choice is in the right column, it must be the CHANNEL's own, not the reference's. When a video plan's visual/voice/topic could be mistaken for the reference creator's, it fails Gate 3.

## The channel identity is three locked choices (set once, brand-defining)

A differentiated channel owns a distinct VISUAL MEDIUM + VOICE REGISTER + TOPIC LANE. Zack D owns "soft educational 3D + calm American narrator + body/nature/hypothetical rotation." We must own something adjacent-but-distinct.

**Visual medium** (open lanes, AI-achievable now, no dominant Shorts channel yet): isometric miniature diorama, claymation-texture stop-motion look, flat-vector (Kurzgesagt-adjacent but distinct palette), noir/silhouette, cinematic photoreal macro. AVOID generic "soft educational 3D" - that reads as Zack D. **Paper-cutout / layered card is NOT an open lane - it is RETIRED (see current-playbook.md); do not pick it.**

**Voice register** (borrow an emotional tone, never a person's actual voice): British-documentary hushed-drop, calm-storyteller, conspiratorial-whisper, warm-and-wry. AVOID the default American TTS narrator (both off-brand and the sound every faceless channel already uses).

**Topic lane** (LOCKED 2026-07-14, see content-direction.md): HIGH-AROUSAL second-person immersion + stakes, NOT detached explainers. The lanes: second-person survival/wonder hypotheticals ("you're caught in a rip current", "you fall into the deepest place on Earth"), hidden-dark-rules of real systems, animal survival drama, safe-morbid body. The saturated "here's how X works" fact-explainer is KILLED (it's what made the first script draft boring). Second-person "you" framing is itself part of the signature.

These three are recorded in the channel plan and reused verbatim across every video, the way a brand reuses a logo. Consistency of an OWN identity is itself a monetization originality signal.

## The differentiation test (per video, before generation)

1. **Swap test:** if you removed the narration, could a viewer tell this from a Zack D video by the visuals alone? If no, the visual medium isn't differentiated enough.
2. **Voice test:** is the narration register the channel's locked voice, not a generic American TTS?
3. **Angle test:** is the topic's angle one the reference creator hasn't led with, or a genuinely fresh framing?
4. **Template test:** is this structurally different enough from the last 3 uploads to not read as a template?

Any "no" is a rework before spending on generation.
