# SaaS Validation: CaptionHQ

**Verdict:** KILL (as standalone product) / REROUTE as AuditHQ feature
**Confidence Score:** 48/100
**Date:** 2026-04-18

## Reasoning

Cross-signal pain is real — Upwork caption gigs at $30-50/post, HN social calendar complaints. But:

- **Competitors:** [Ocoya](https://ocoya.com), [Postiz](https://postiz.com), [Hootsuite AI](https://hootsuite.com), [Later](https://later.com), [Buffer](https://buffer.com), [SocialPilot](https://socialpilot.co), [ContentStudio](https://contentstudio.io) — all paid, all established
- **No moat:** "AI captions + scheduling" is a commodity. Adam has no data advantage, no distribution advantage specific to this category
- **Adjacency flag:** CaptionHQ could be a **feature inside AuditHQ** — the reputation audit already scans social presence. Extending it with "AI generate post-audit content" would:
  - Reuse AuditHQ's customer base (higher adjacency = 8/10)
  - Provide expansion revenue to existing AuditHQ customers
  - Not require new distribution

## Verdict

**KILL as standalone product.** Reroute to AuditHQ backlog as a post-audit content generator feature (to be built only after AuditHQ hits $5K/mo per portfolio gate).

## Focus Instead On
AuditHQ primary revenue push. Add "content generator from audit findings" to AuditHQ feature backlog, not as separate product.
